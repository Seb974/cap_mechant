# Launch

## Download
First of all download or clone the project's files.
```shell
git clone https://github.com/YenByNigao/codac-pff.git
```
## Installation
```shell
cd codac-pff
cd ap_hero
composer install
php bin/console doctrine:database:create
php bin/console doctrine:migrations:migrate
```
## Running the project
```shell
php server:run
```
